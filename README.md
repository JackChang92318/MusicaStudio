## 操作方法
<h4>從放置sample的區域使用滑鼠左鍵拖曳進入編曲區域，並且按照所要的聲音pattern進行編曲</h4>
<h4>可以透過sidechain功能強調鼓點的節奏，也可以透過equalizer對聲音進行低中高三種頻率進行過濾</h4>
<h4>可以透過旋鈕控制單軌的音量大小，也可以直接靜音</h4>
